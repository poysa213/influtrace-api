# Katch Api

This is the backend repository for the Katch Api.

## Installation

First install the dependencies of the npm:

```bash
npm i
```

## Running the development server

To run the development server, use the following command:

```bash
npm run dev
```

## Building for production

You can build the project for production by running the following command:

```bash
npm run build
```

## Run the production server

To run the production server, use the following command:

```bash
npm run server
```

Or following command will build the project and run the production server on single command:  

```bash
npm start
```
